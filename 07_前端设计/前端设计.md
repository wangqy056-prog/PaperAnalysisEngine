```

┌──────────────┬──────────────────────────────────────────────┐
│              │                                              │

│   Dashboard  │   │      搜索结果页 / 论文列表页         │   │
│   Search     │   └──────────────────────────────────────┘   │
│   Paper      │                                              │
│   Details    │   ┌──────────────────────────────────────┐   │
│   Analysis   │   │      论文详情页                      │   │
│   Combos     │   └──────────────────────────────────────┘   │
│   Roadmap    │                                              │
│   AI Chat    │   ┌──────────────────────────────────────┐   │
│   Settings   │   │      组合分析页                      │   │

│              │   │      AI对话页                        │   │
│              │   └──────────────────────────────────────┘   │
│              │                                              │
```

### 1.2 核心页面

| 页面 | 功能 | 核心组件 |
|------|------|---------|
| **Dashboard** | 首页仪表盘 | 热门论文、评级分布、趋势图表 |
| **Search** | 论文搜索 | 搜索框、筛选器、结果列表 |
| **Paper Details** | 论文详情 | 论文卡片、评级、标签、分析报告 |
| **Analysis** | 深度分析 | 评级雷达图、商业化预测、风险评估 |
| **Combos** | 组合分析 | 组合列表、协同度图表、技术路线图 |
| **AI Chat** | AI对话 | 聊天界面、RAG检索结果 |
| **Settings** | 设置 | API配置、评级参数调整 |

## 三、技术实现

### 3.1 前端技术栈

| 分类 | 技术 | 版本 | 用途 |
|------|------|------|------|
| **框架** | React | 18+ | 主框架 |
| **构建** | Vite | 6+ | 构建工具 |
| **语言** | TypeScript | 5+ | 类型安全 |
| **UI组件** | shadcn/ui | latest | 组件库 |
| **图表** | ECharts | 5+ | 数据可视化 |
| **图谱** | React-Force-Graph | latest | 知识图谱可视化 |
| **状态管理** | Zustand | latest | 轻量级状态管理 |
| **路由** | React Router | 6+ | 路由管理 |
| **API调用** | Axios | 1+ | HTTP客户端 |

### 3.3 API接口设计

```typescript
// 论文搜索
GET /api/papers/search?query=xxx&limit=100

// 获取论文详情
GET /api/papers/:id

// 获取论文分析
GET /api/papers/:id/analysis

// 获取论文组合
GET /api/papers/:id/combos

// 获取技术路线图
GET /api/roadmap?topic=xxx

// AI对话
POST /api/chat
{
  "message": "xxx",
  "context": ["paper_id1", "paper_id2"]
}

// 获取评级体系
GET /api/ratings/criteria

// 更新评级参数
PUT /api/ratings/criteria
```

## 四、设计规范

### 4.1 颜色规范

| 颜色 | 用途 |
|------|------|
| **金色 #FFD700** | S级论文 |
| **绿色 #4CAF50** | A级论文 |
| **蓝色 #2196F3** | B级论文 |
| **橙色 #FF9800** | C级论文 |
| **红色 #F44336** | D级论文 |
| **紫色 #9C27B0** | AI分析 |
| **灰色 #9E9E9E** | 普通文本 |

### 4.2 响应式设计

- **桌面端**：三栏布局（侧边栏+内容区+图谱）
- **平板端**：两栏布局（侧边栏+内容区）
- **移动端**：单栏布局（底部导航）
