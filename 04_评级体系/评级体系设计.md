|------|------|---------|---------|

## 二、量化评级公式

> **设计原则**：每个维度必须有明确的数学公式，避免模糊描述
> **参数说明**：以下公式中的具体权重、阈值、系数等均在后端 config.py 中配置

### 2.1 学术影响力评级

**核心公式**：

```
学术影响力 = Σ(引用量 × 时间衰减 × 期刊权重) / 标准化因子

展开：
AI = (C × T_decay × J_weight + R_rank × W_rank + SS_score × W_ss) / Z

其中：
• C = 引用量（原始值）
• T_decay = e^(-λ × (当前年份 - 发表年份))，λ = <见后端 config.py>（时间衰减系数）
• J_weight = 期刊权重（各期刊权重见 config.py）
• R_rank = 领域排名百分位（0-1），转换为评分 = 100 × (1 - rank)
• W_rank = <见后端 config.py>（领域排名权重）
• SS_score = Semantic Scholar影响力评分（0-100）
• W_ss = <见后端 config.py>（SS评分权重）
• Z = <见后端 config.py>（标准化因子，将结果缩放到0-100）
```

**Python 实现**：

```python
import math

def calculate_academic_impact(paper):
    """学术影响力量化公式"""
    
    current_year = 2026
    paper_year = paper.year or current_year
    
    # 1. 引用量 × 时间衰减 × 期刊权重
    citations = paper.citations or 0
    lambda_decay = <见后端 config.py>  # 时间衰减系数
    time_decay = math.exp(-lambda_decay * (current_year - paper_year))
    
    # 期刊权重字典（见 config.py）
    journal_weights = <见后端 config.py>
    j_weight = journal_weights.get(paper.journal, journal_weights.get("default", 0.5))
    
    citation_score = citations * time_decay * j_weight
    
    # 2. 领域排名评分（假设有API提供）
    # R_rank = get_field_rank(paper)  # 返回0-1的百分位
    # rank_score = 100 * (1 - R_rank) if R_rank else <见后端 config.py>
    
    # 简化：用引用量在同领域排名估算
    rank_threshold = <见后端 config.py>
    rank_coeff = <见后端 config.py>
    rank_score = min(rank_threshold, citations / rank_coeff)
    
    # 3. Semantic Scholar评分（如果有）
    ss_score = paper.impact_score or <见后端 config.py>  # 默认值
    
    # 4. 综合计算
    Z = <见后端 config.py>  # 标准化因子
    W_rank = <见后端 config.py>  # 领域排名权重
    W_ss = <见后端 config.py>  # SS评分权重
    ai = (citation_score + rank_score * W_rank + ss_score * W_ss) / Z * 100
    
    return min(100, max(0, ai))
```

### 2.2 商业转化潜力评级

**核心公式**：

```
商业潜力 = f(TRL, 专利数, 投资热度, 市场增速) × 时间修正

展开：
CP = (TRL × W_trl + Patent_score + Market_score + Competition_score) × T_factor

其中：
• TRL = 技术成熟度等级（1-9）
• W_trl = <见后端 config.py>（TRL转换系数）
• Patent_score = min(W_patent_max, ln(专利数+1) × W_patent_coeff)
• W_patent_max = <见后端 config.py>（专利评分上限）
• W_patent_coeff = <见后端 config.py>（专利评分系数）
• Market_score = 根据市场规模区间评分（阈值见 config.py）
• Competition_score = 根据竞争格局评分（评分标准见 config.py）
• T_factor = 时间修正因子（TRL>=7时×<见后端 config.py>，TRL<=3时×<见后端 config.py>）
```

**Python 实现**：

```python

        base_time = shorten_time(base_time, <见后端 config.py>)
    
    return {
        "base_time": base_time,
        "adjusted_time": base_time,
        "factors": factors,
        "confidence": calculate_confidence(factors)
    }
```

## 四、风险评估体系

### 4.1 风险类型

| 风险类型 | 评估指标 | 风险等级 |
|---------|---------|---------|
| **技术风险** | 技术成熟度、可靠性、可扩展性 | 低/中/高/极高 |
| **商业化风险** | 市场接受度、竞争格局、成本结构 | 低/中/高/极高 |
| **伦理风险** | 数据隐私、AI偏见、社会影响 | 低/中/高/极高 |
| **专利风险** | IP布局、侵权风险、专利诉讼 | 低/中/高/极高 |
| **监管风险** | 政策法规、行业标准、审批流程 | 低/中/高/极高 |

### 4.2 风险评估算法

```python
def assess_risk(paper):
    """评估风险等级"""
    
    risks = {}
    
    # 技术风险
    tech_risk = assess_tech_risk(paper)
    risks["技术风险"] = tech_risk
    
    # 商业化风险
    commercial_risk = assess_commercial_risk(paper)
    risks["商业化风险"] = commercial_risk
    
    # 伦理风险
    ethical_risk = assess_ethical_risk(paper)
    risks["伦理风险"] = ethical_risk
    
    # 专利风险
    patent_risk = assess_patent_risk(paper)
    risks["专利风险"] = patent_risk
    
    # 综合风险等级
    overall_risk = calculate_overall_risk(risks)
    
    return {
        "risks": risks,
        "overall_risk": overall_risk,
        "risk_summary": generate_risk_summary(risks)
    }
```
