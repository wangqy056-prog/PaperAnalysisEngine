## 二、量化评级公式

> **设计原则**：每个维度必须有明确的数学公式，避免模糊描述

### 2.1 学术影响力评级

**核心公式**：

```
学术影响力 = Σ(引用量 × 时间衰减 × 期刊权重) / 标准化因子

展开：
AI = (C × T_decay × J_weight + R_rank × 25 + SS_score × 20) / Z

其中：
• C = 引用量（原始值）
• T_decay = e^(-λ × (当前年份 - 发表年份))，λ = 0.1（10年衰减一半）
• J_weight = 期刊权重（Nature=1.5, Science=1.5, Cell=1.5, 子刊=1.2, 普通SCI=1.0, 其他=0.5）
• R_rank = 领域排名百分位（0-1），转换为评分 = 100 × (1 - rank)
• SS_score = Semantic Scholar影响力评分（0-100）
• Z = 标准化因子 = 150（将结果缩放到0-100）
```

**Python 实现**：

```python
import math

def calculate_academic_impact(paper):
    """学术影响力量化公式"""
    
    current_year = 2026
    paper_year = paper.year or current_year
    
    # 1. 引用量 × 时间衰减 × 期刊权重
    citations = paper.citations or 0
    time_decay = math.exp(-0.1 * (current_year - paper_year))  # 10年衰减一半
    
    # 期刊权重字典
    journal_weights = {
        "Nature": 1.5, "Science": 1.5, "Cell": 1.5,
        "Nature Machine Intelligence": 1.2, "Nature Methods": 1.2,
        "IEEE Transactions": 1.0, "ACM": 1.0,
        "default": 0.5
    }
    j_weight = journal_weights.get(paper.journal, 0.5)
    
    citation_score = citations * time_decay * j_weight
    
    # 2. 领域排名评分（假设有API提供）
    # R_rank = get_field_rank(paper)  # 返回0-1的百分位
    # rank_score = 100 * (1 - R_rank) if R_rank else 25
    
    # 简化：用引用量在同领域排名估算
    rank_score = min(25, citations / 100 * 5)  # 简化公式
    
    # 3. Semantic Scholar评分（如果有）
    ss_score = paper.impact_score or 50
    
    # 4. 综合计算
    Z = 150  # 标准化因子
    ai = (citation_score + rank_score * 25 + ss_score * 0.2) / Z * 100
    
    return min(100, max(0, ai))
```

### 2.2 商业转化潜力评级

**核心公式**：

```
商业潜力 = f(TRL, 专利数, 投资热度, 市场增速) × 时间修正

展开：
CP = (TRL × 4 + Patent_score + Market_score + Competition_score) × T_factor

其中：
• TRL = 技术成熟度等级（1-9）
• Patent_score = min(25, ln(专利数+1) × 10)
• Market_score = 市场规模评分（>10B=20, >1B=15, >100M=10, 其他=5）
• Competition_score = 竞争格局评分（低竞争=15, 中=10, 高=5）
• T_factor = 时间修正因子（TRL>=7时×1.2，TRL<=3时×0.8）
```

**Python 实现**：

```python
import math

def calculate_commercial_potential(paper, patent_count=0, market_size=0, competition="medium"):
    """商业转化潜力量化公式"""
    
    # 1. TRL评分（假设有估算函数）
    trl = estimate_trl(paper)  # 返回1-9
    trl_score = trl * 4  # TRL 9 → 36分
    
    # 2. 专利评分
    patent_score = min(25, math.log(patent_count + 1) * 10)
    
    # 3. 市场规模评分
    if market_size >= 10_000_000_000:  # >100亿
        market_score = 20
    elif market_size >= 1_000_000_000:  # >10亿
        market_score = 15
    elif market_size >= 100_000_000:  # >1亿
        market_score = 10
    else:
        market_score = 5
    
    # 4. 竞争格局评分
    competition_scores = {"low": 15, "medium": 10, "high": 5}
    comp_score = competition_scores.get(competition, 10)
    
    # 5. 时间修正因子
    if trl >= 7:
        t_factor = 1.2  # 快商业化，加权
    elif trl <= 3:
        t_factor = 0.8  # 远商业化，减权
    else:
        t_factor = 1.0
    
    # 6. 综合计算
    cp = (trl_score + patent_score + market_score + comp_score) * t_factor
    
    return min(100, max(0, cp))

def estimate_trl(paper):
    """估算TRL等级（基于论文特征）"""
    # 简化估算：基于关键词和发表年份
    if paper.year >= 2024 and "prototype" in paper.abstract.lower():
        return 6  # 技术验证阶段
    elif paper.year >= 2022 and "experiment" in paper.abstract.lower():
        return 4  # 实验验证阶段
    elif "theory" in paper.abstract.lower() or "framework" in paper.abstract.lower():
        return 2  # 基础研究阶段
    else:
        return 5  # 默认中等
```

### 2.3 创新指数评级

**核心公式**：

```
创新指数 = (原创性 × 35 + 方法创新 × 25 + 理论贡献 × 20 + 实验验证 × 20)

其中各子评分通过LLM分析得出（0-1的系数）：
• 原创性 = LLM评分（"这篇论文是否提出全新概念？"）→ 输出0-1
• 方法创新 = LLM评分（"这篇论文是否有方法论创新？"）→ 输出0-1
• 理论贡献 = LLM评分（"这篇论文是否有理论贡献？"）→ 输出0-1
• 实验验证 = 检查关键词（"validated", "tested", "experiment"）→ 输出0-1
```

**Python 实现**：

```python
def calculate_innovation_index(paper, llm_scores=None):
    """创新指数量化公式"""
    
    if llm_scores:
        # 使用LLM分析结果
        originality = llm_scores.get("originality", 0.5)
        method_innovation = llm_scores.get("method_innovation", 0.5)
        theory_contribution = llm_scores.get("theory_contribution", 0.5)
    else:
        # 简化估算：基于关键词
        abstract_lower = paper.abstract.lower() if paper.abstract else ""
        
        originality = 0.8 if "novel" in abstract_lower or "new" in abstract_lower else 0.5
        method_innovation = 0.7 if "method" in abstract_lower or "approach" in abstract_lower else 0.4
        theory_contribution = 0.6 if "theory" in abstract_lower or "framework" in abstract_lower else 0.3
    
    # 实验验证评分（基于关键词）
    abstract_lower = paper.abstract.lower() if paper.abstract else ""
    if "validated" in abstract_lower or "tested" in abstract_lower or "experiment" in abstract_lower:
        experimental_validation = 0.8
    elif "simulated" in abstract_lower or "evaluated" in abstract_lower:
        experimental_validation = 0.5
    else:
        experimental_validation = 0.2
    
    # 综合计算
    ii = originality * 35 + method_innovation * 25 + theory_contribution * 20 + experimental_validation * 20
    
    return min(100, max(0, ii))
```

### 2.4 可复现性评级

**核心公式**：

```
可复现性 = Σ(各项检查得分)

检查项（布尔值，有则得分）：
• 数据公开: +30 (检查是否有 GitHub/Data链接)
• 代码公开: +25 (检查是否有 GitHub链接)
• 实验细节: +25 (检查摘要中是否有详细实验描述)
• 可复现标记: +20 (检查是否有 "replicable" 关键词)
```

**Python 实现**：

```python
def calculate_reproducibility(paper):
    """可复现性量化公式"""
    
    score = 0
    abstract_lower = paper.abstract.lower() if paper.abstract else ""
    
    # 1. 数据公开检查
    if paper.pdf_url and "github" in abstract_lower:
        score += 30
    elif "data available" in abstract_lower or "dataset" in abstract_lower:
        score += 15
    
    # 2. 代码公开检查
    if "github.com" in abstract_lower or "code available" in abstract_lower:
        score += 25
    elif "implementation" in abstract_lower:
        score += 12
    
    # 3. 实验细节检查
    if "method" in abstract_lower and "parameter" in abstract_lower:
        score += 25
    elif "experiment" in abstract_lower:
        score += 15
    
    # 4. 可复现标记检查
    if "replicable" in abstract_lower or "reproducible" in abstract_lower:
        score += 20
    elif "open source" in abstract_lower:
        score += 10
    
    return min(100, score)
```

### 2.5 组合价值评级

**核心公式**：

```
组合价值 = (方法论互补性 × 30 + 场景扩展 × 25 + 技术叠加 × 25 + 商业协同 × 20)

注：此评分需要结合其他论文分析，第一阶段可设为默认值50
```

**Python 实现**：

```python
def calculate_combo_value(paper, related_papers=None):
    """组合价值量化公式"""
    
    if not related_papers:
        # 第一阶段默认值
        return 50
    
    # 简化估算：基于论文关键词多样性
    abstract_lower = paper.abstract.lower() if paper.abstract else ""
    
    # 1. 方法论互补性（检查是否有多种方法）
    methods = ["machine learning", "statistical", "experimental", "simulation"]
    method_count = sum(1 for m in methods if m in abstract_lower)
    complementarity = min(1, method_count / 2)
    
    # 2. 场景扩展（检查是否有多个应用场景）
    scenarios = ["medical", "finance", "manufacturing", "energy", "transportation"]
    scenario_count = sum(1 for s in scenarios if s in abstract_lower)
    scenario_expansion = min(1, scenario_count / 2)
    
    # 3. 技术叠加（检查是否有技术组合）
    if "combined" in abstract_lower or "integrated" in abstract_lower:
        synergy_effect = 0.8
    else:
        synergy_effect = 0.4
    
    # 4. 商业协同（检查是否有商业应用）
    if "application" in abstract_lower or "deployment" in abstract_lower:
        commercial_synergy = 0.7
    else:
        commercial_synergy = 0.3
    
    # 综合计算
    cv = complementarity * 30 + scenario_expansion * 25 + synergy_effect * 25 + commercial_synergy * 20
    
    return min(100, max(0, cv))
```

## 三、商业化时间预测模型

### 3.1 TRL技术成熟度曲线

```
TRL 1-3: 基础研究阶段 → 预测商业化时间: 5-10年
TRL 4-6: 技术开发阶段 → 预测商业化时间: 3-5年  
TRL 7-8: 验证示范阶段 → 预测商业化时间: 1-3年
TRL 9: 商业化阶段 → 预测商业化时间: 0-1年
```

| **专利风险** | IP布局、侵权风险、专利诉讼 | 低/中/高/极高 |
