    关键词：{paper.keywords}

## 三、知识图谱关系

### 3.1 关系类型

| 关系 | 源节点 | 目标节点 | 属性 |
|------|--------|----------|------|
| CITES | Paper | Paper | citation_count, year |
| WRITTEN_BY | Paper | Author | contribution_rank |
| PUBLISHED_IN | Paper | Journal | volume, issue, page |
| AFFILIATED_WITH | Author | Institution | start_year, end_year |
| BELONGS_TO | Paper | Category | confidence |
| HAS_TAG | Paper | Tag | confidence, relevance |
| SYNERGY_WITH | Paper | Paper | synergy_score, description |
| DEPENDS_ON | Paper | Paper | dependency_type |
| FOLLOWS | Paper | Paper | temporal_order |

### 3.2 图谱查询示例

```cypher
// 查找某篇论文引用的所有论文
MATCH (p:Paper {doi: "xxx"})-[:CITES]->(ref:Paper)
RETURN ref.title, ref.year, ref.citations

// 查找某领域的核心论文（按引用排序）
MATCH (p:Paper)-[:BELONGS_TO]->(c:Category {name: "AI"})
RETURN p.title, p.citations
ORDER BY p.citations DESC
LIMIT 20

// 查找论文的协同论文
MATCH (p:Paper {doi: "xxx"})-[:SYNERGY_WITH]->(other:Paper)
RETURN other.title, other.synergy_score
ORDER BY other.synergy_score DESC
```
