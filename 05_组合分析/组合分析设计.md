# 论文组合分析引擎

## 一、组合分析概念

### 1.1 什么是论文组合效应

论文组合效应指**两篇或多篇论文的思想、方法或技术相互结合，产生大于单独使用时价值的现象**。

类比药物组合（synergy），论文也可以有：
- **互补效应**：方法论互补，A的缺陷由B弥补
- **叠加效应**：技术叠加，产生新的突破
- **场景扩展**：A在场景X有效，B在场景Y有效，组合后覆盖X+Y+Z
- **创新催化**：A的理论 + B的实验方法 = C的新发现

### 1.2 组合价值公式

```
组合价值 = Σ(单篇价值) × 协同因子 × 场景扩展因子 × 时间因子

协同因子：基于方法论互补性计算（范围见 config.py）
场景扩展因子：基于应用场景覆盖率计算（范围见 config.py）
时间因子：考虑技术演进阶段（范围见 config.py）
```

    # 2. 计算两两协同度
    combos = []
    synergy_threshold = <见后端 config.py>  # 协同度阈值

## 三、组合类型

### 3.1 组合模式

| 组合模式 | 描述 | 示例 |
|---------|------|------|
| **理论+实验** | 理论论文提供框架，实验论文验证或应用 | 量子计算理论 + 量子纠错实验 |
| **方法+应用** | 方法论论文提供工具，应用论文展示落地 | 扩散模型理论 + 图像生成应用 |
| **技术+场景** | 技术论文提供方案，场景论文提供需求 | 固态电池技术 + 电动汽车应用 |
| **基础+工程** | 基础研究提供原理，工程论文提供实现 | 钙钛矿理论 + 太阳能电池工程 |
| **跨领域融合** | 不同领域论文交叉产生新方向 | AI + 生物 + 材料科学 |

### 5.1 空白识别算法

```python
def identify_research_gaps(topic, focus_area=None):
    """识别研究空白"""
    
    # 1. 获取现有研究图谱
    papers = search_papers(topic)
    
    # 2. 分析研究热点分布
    hot_topics = analyze_hot_topics(papers)
    
    # 3. 分析研究趋势
    trends = analyze_trends(papers)
    
    # 4. 识别空白区域
    gaps = []
    
    # 方法1：基于主题聚类
    clusters = cluster_topics(papers)
    MIN_PAPERS = <见后端 config.py>  # 最小论文数阈值
    for cluster in clusters:
        if len(cluster.papers) < MIN_PAPERS:
            gaps.append({
                "type": "understudied",
                "topic": cluster.topic,
                "reason": "研究数量不足",
                "potential": calculate_potential(cluster)
            })
    
    # 方法2：基于趋势分析
    ACCELERATION_THRESHOLD = <见后端 config.py>  # 增长加速阈值
    PAPER_COUNT_LIMIT = <见后端 config.py>  # 论文数量限制
    for trend in trends:
        if trend.acceleration > ACCELERATION_THRESHOLD and trend.paper_count < PAPER_COUNT_LIMIT:
            gaps.append({
                "type": "emerging",
                "topic": trend.topic,
                "reason": "增长迅速但研究不足",
                "potential": calculate_potential(trend)
            })
    
    # 方法3：基于组合分析
    combos = find_unexplored_combos(papers)
    for combo in combos:
        gaps.append({
            "type": "combinatorial",
            "topic": combo.description,
            "reason": "两种技术尚未结合",
            "potential": combo.synergy_score
        })
    
    return sorted(gaps, key=lambda x: x["potential"], reverse=True)
```

def suggest_patent_strategy(papers):
    """基于论文组合的专利策略建议"""
    
    # 1. 分析现有专利布局
    existing_patents = analyze_existing_patents(papers)
    
    # 2. 识别专利空白
    patent_gaps = identify_patent_gaps(papers, existing_patents)
    
    # 3. 建议专利申请方向
    suggestions = []
    
    for gap in patent_gaps:
        suggestions.append({
            "patent_type": gap.type,
            "claim_scope": gap.claim_scope,
            "priority": gap.priority,
            "timing": gap.timing,
            "risk_assessment": gap.risk_assessment
        })
    
    # 4. 建议专利组合
    combo_suggestions = suggest_patent_combos(papers)
    
    return {
        "patent_gaps": patent_gaps,
        "suggestions": suggestions,

    }
