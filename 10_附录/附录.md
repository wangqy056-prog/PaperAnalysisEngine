```bash
cd frontend
```

### 3.2 Neo4j查询示例

```cypher
// 创建论文节点
CREATE (p:Paper {
    id: "123",
    title: "xxx",
    year: 2026,
    citations: 100
})

// 创建引用关系
MATCH (p1:Paper {id: "123"}), (p2:Paper {id: "456"})
CREATE (p1)-[:CITES]->(p2)

// 查询某领域高引用论文
MATCH (p:Paper)-[:BELONGS_TO]->(c:Category {name: "AI"})
RETURN p.title, p.citations
ORDER BY p.citations DESC
LIMIT 10
```

### 3.3 API调用示例

